   ```bash
   # Windows (管理员权限)
   winget install Kitware.CMake
   winget install Microsoft.VisualStudio.2022.BuildTools
   
   # 或者安装完整的Visual Studio
   winget install Microsoft.VisualStudio.2022.Community
   ```

2. **安装Qt6** (用于GUI):
